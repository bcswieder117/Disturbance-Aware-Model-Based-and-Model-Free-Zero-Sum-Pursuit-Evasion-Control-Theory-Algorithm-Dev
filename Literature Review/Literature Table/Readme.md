# Literature Review Table

This directory contains the literature review table developed for the thesis:

**Disturbance-Aware Model-Based and Model-Free Zero-Sum Pursuit-Evasion Control: Theory, Algorithm Development, and MATLAB Validation**

The table organizes the primary references that support the thesis theory, algorithm development, disturbance-aware extensions, MATLAB validation, and pursuit-evasion interpretation.

The current version is structured as a focused research map rather than a general bibliography. It prioritizes recent work while retaining the foundational references needed to explain the origin of the algorithms and mathematical results used in the thesis.
