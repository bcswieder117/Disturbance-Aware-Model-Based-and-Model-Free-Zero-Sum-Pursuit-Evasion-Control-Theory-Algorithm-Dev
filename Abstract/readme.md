# Disturbance-Aware Model-Based and Model-Free Zero-Sum Pursuit–Evasion Control

> **Project Status:** Active Master of Science thesis project at Tennessee Technological University. This repository contains the reproducible MATLAB implementations, numerical experiments, figures, literature documentation, and supporting materials developed for the thesis.

## Thesis Title

**Disturbance-Aware Model-Based and Model-Free Zero-Sum Pursuit–Evasion Control: Theory, Algorithm Development, and MATLAB Validation**

## Overview

This research develops a unified disturbance-aware framework for a discrete-time, two-player, zero-sum pursuit–evasion game. The project lies at the intersection of zero-sum optimal control, game-theoretic multi-agent systems, and model-free quadratic Q-learning, with a focus on disturbance-aware pursuit–evasion control and MATLAB-based validation.

The pursuer is represented as the minimizing player, while the evader is represented as the maximizing player. Both agents act through separate control channels and operate under a shared linear-quadratic game objective.

The thesis studies four established solution methods:

1. Model-based generalized Riccati policy iteration.
2. Model-based generalized Riccati value iteration.
3. Model-free quadratic Q-learning policy iteration.
4. Model-free quadratic Q-learning value iteration.

The original policy-iteration, value-iteration, and Q-learning structures are preserved. The research contribution is the addition of a common disturbance-aware extension involving persistent command-channel offsets, offset estimation, affine command compensation, and matched MATLAB validation.

## Abstract

Autonomous and adversarial multi-agent systems require control decisions that remain effective when the inputs applied to a system differ from the commands generated by the controller. This thesis investigates disturbance-aware control for a discrete-time, two-player, zero-sum pursuit–evasion game and develops a common framework for extending established model-based and model-free solution methods without replacing their original policy-iteration, value-iteration, or Q-learning structures. The pursuer is represented as the minimizing player, while the evader is represented as the maximizing player within a linear-quadratic dynamic game. The nominal saddle-point policies are characterized through the generalized algebraic Riccati equation and an equivalent quadratic action-value representation.

Four established methods are considered: model-based generalized Riccati policy iteration, model-based generalized Riccati value iteration, model-free quadratic Q-learning policy iteration, and model-free quadratic Q-learning value iteration. Each method is extended to include persistent command-channel offsets affecting both the pursuer and evader. The offsets represent repeatable differences between commanded and realized inputs. Offset estimates are obtained from transition residuals or measured command mismatch, depending on the information structure of the algorithm. Affine compensation is then applied by subtracting the estimated offsets from the commanded control actions. The resulting residual-bias dynamics directly relate the compensated closed-loop response to the remaining estimation error and establish exact compensation as a verification case.

MATLAB studies evaluate nominal operation, uncompensated command offsets, estimated-offset compensation, and exact compensation under matched system matrices, game weights, disturbances, initial conditions, and numerical criteria. The validation includes policy and value convergence, generalized Riccati and Bellman residuals, closed-loop spectral radius, saddle-point curvature, matrix conditioning, feature-rank requirements, bias-estimation error, state and input trajectories, and cumulative game cost. The thesis does not introduce new base algorithms. Its contribution is the unified disturbance-aware extension and matched evaluation of four established zero-sum control methods within a common pursuit–evasion framework. Pursuit–evasion provides the application interpretation, while nonlinear autonomous-ground-vehicle and QCar implementation are retained as future extensions rather than completed hardware validation.

## Common Game Formulation

The nominal discrete-time zero-sum system is represented as:

```text
x(k+1) = A*x(k) + BP*uP(k) + BE*uE(k)
```

where:

- `x(k)` is the game state;
- `uP(k)` is the pursuer input;
- `uE(k)` is the evader input;
- `BP` is the pursuer input matrix;
- `BE` is the evader input matrix.

The pursuer attempts to minimize the quadratic game cost, while the evader attempts to maximize the same cost.

A representative infinite-horizon objective is:

```text
J = sum over k of:

    x(k)'*Q*x(k)
  + uP(k)'*RP*uP(k)
  - uE(k)'*RE*uE(k)
```

The nominal feedback policies are represented as:

```text
uP(k) = L*x(k)
uE(k) = K*x(k)
```

The corresponding nominal closed-loop system is:

```text
x(k+1) = (A + BP*L + BE*K)*x(k)
```

## Disturbance-Aware Extension

The primary disturbance model consists of persistent command-channel offsets affecting the realized pursuer and evader inputs:

```text
uP_actual(k) = uP_command(k) + bP
uE_actual(k) = uE_command(k) + bE
```

where:

- `bP` is the pursuer command offset;
- `bE` is the evader command offset.

Without compensation, the disturbed system becomes:

```text
x(k+1) = (A + BP*L + BE*K)*x(k)
         + BP*bP
         + BE*bE
```

The estimated offsets are denoted by:

```text
bP_hat
bE_hat
```

Affine compensation is applied through:

```text
uP_command(k) = L*x(k) - bP_hat
uE_command(k) = K*x(k) - bE_hat
```

The compensated dynamics are therefore:

```text
x(k+1) = (A + BP*L + BE*K)*x(k)
         + BP*(bP - bP_hat)
         + BE*(bE - bE_hat)
```

When the offset estimates are exact, the nominal closed-loop dynamics are recovered.

## Four Algorithmic Components

### Model-Based Policy Iteration

The model-based policy-iteration method alternates between policy evaluation and saddle-policy improvement using the known system matrices.

The disturbance-aware extension is applied without replacing the original policy-evaluation or policy-improvement equations.

### Model-Based Value Iteration

The model-based value-iteration method repeatedly updates the zero-sum value matrix through a generalized Riccati mapping and recovers the pursuer and evader gains from the current value estimate.

The same command-offset model, estimation structure, and compensation law are used for comparison with policy iteration.

### Model-Free Q-Learning Policy Iteration

The model-free policy-iteration method estimates a quadratic action-value function from sampled state, input, cost, and successor-state data.

The joint state-action vector is represented as:

```text
z(k) = [x(k); uP(k); uE(k)]
```

The quadratic Q-function is represented as:

```text
QH(z(k)) = z(k)'*H*z(k)
```

The pursuer and evader policies are recovered from the appropriate matrix partitions of `H`.

### Model-Free Q-Learning Value Iteration

The model-free value-iteration method uses sampled Bellman updates to estimate successive quadratic action-value functions and recover the corresponding pursuer and evader policies.

This method is evaluated using the same system, command offsets, compensation cases, and numerical checks as the other three algorithms.

## MATLAB Validation Cases

Each algorithm is evaluated under four matched simulation cases:

1. **Nominal Operation**  
   No command-channel offsets are applied.

2. **Uncompensated Command Offsets**  
   Persistent pursuer and evader offsets are applied without correction.

3. **Estimated Offset Compensation**  
   Estimated offsets are subtracted from the commanded inputs.

4. **Exact Offset Compensation**  
   The true offsets are used to verify recovery of the nominal closed-loop response.

The same system matrices, game weights, initial conditions, simulation horizon, command offsets, and evaluation criteria are used across the four algorithms whenever the algorithmic information structures permit a direct comparison.

## Pursuit–Evasion Interpretation

The zero-sum dynamic game is interpreted as a pursuit–evasion interaction.

The pursuer seeks to reduce a game-state error or drive the system toward a desired capture-related condition. The evader seeks to increase the same cost or resist the pursuer’s objective.

The present framework uses a reduced-order linear game so that the four algorithms and the disturbance-aware extension can be analyzed under common mathematical and numerical conditions.

Nonlinear autonomous-ground-vehicle models, kinematic bicycle dynamics, steering and acceleration constraints, QLabs simulation, and QCar experiments are retained as future implementation pathways.
